# Grand Line Adventure 🏴‍☠️

**One Piece × Disney — Open World 3D**  
Built with Three.js r134 — aucune dépendance externe.

## Structure
```
grand-line/
├── index.html       ← page principale
├── js/
│   ├── three.min.js ← Three.js r134 (inclus)
│   └── game.js      ← moteur du jeu
└── README.md
```

## Déployer sur GitHub Pages
1. Crée un repo GitHub et push ce dossier
2. Va dans **Settings → Pages → Source → main / root**
3. Ton jeu sera accessible sur `https://[ton-pseudo].github.io/[repo-name]/`

## Contrôles
| Mobile | Action |
|--------|--------|
| Joystick gauche | Courir / Naviguer |
| Glisser écran droit | Pivoter la caméra |
| 🍖 | Sprint |
| MONTER | Embarquer sur un bateau/monture |
| ⚡ | Gomu Gomu attack |

| Clavier | Action |
|---------|--------|
| WASD / Flèches | Bouger |
| Shift | Sprint |
| E | Embarquer/Débarquer |
| Espace | Attaque |
| Échap | Pause |

## Fonctionnalités
- 🗺️ Archipel open world avec 6 îles
- 🏴‍☠️ Luffy jouable en 3D style Disney/Pixar
- ⛵ 5 véhicules : bateaux & montures
- 🏙️ Ville pirate avec bâtiments pagode
- 🌊 Océan animé, mouettes volantes
- 🗺️ Minimap en temps réel
- 📱 Contrôles tactiles complets
